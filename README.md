Sistema de Verificación Vehicular en C

Programa en C que simula un sistema de verificación vehicular utilizando colas dinámicas.

## ¿Qué hace el programa?

- Los autos ingresan a una unifila.
- Se distribuyen a 5 líneas según el último número de su placa.
- Se atienden autos de manera aleatoria.
- Se puede visualizar el contenido de cada cola.

## Estructuras utilizadas

- Cola (FIFO)
- Listas enlazadas
- Memoria dinámica con malloc y free

## Compilación

gcc main.c -o programa
./programa
